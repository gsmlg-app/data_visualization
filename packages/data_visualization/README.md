## What's Included

This meta-package bundles the entire data visualization suite:

- **Charts**: Line, Area, Bar, Scatter, XY Chart, Heatmap, Network graphs
- **Geographic**: Maps with projections, GeoJSON support, 450+ prebuilt country maps
- **Statistical**: Delaunay triangulation, Voronoi diagrams, hierarchical layouts
- **Components**: Axes, Grids, Legends, Tooltips, Annotations
- **Interactions**: Zoom, Pan, Brush selection, Drag
- **Utilities**: Scales, Shapes, Gradients, Patterns, Mock data generators

## Individual Packages

For a smaller bundle size, import only the packages you need:
- `dv_xychart` - XY charts (line, bar, area, scatter)
- `dv_map` - Geographic maps with Natural Earth data
- `dv_heatmap` - Heatmap visualizations
- `dv_network` - Network/graph visualizations
