# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

### Initial Release

- **dv_vendor**: Core utility functions - array operations (min, max, extent, sum, mean, median, quantile, histogram) and interpolation (number, RGB, date, basis)
- **dv_point**: 2D Point class with vector operations, transformations, and list extensions
- **dv_scale**: Scale functions - linear, log, power, symlog, time, band, point, ordinal, quantize, quantile, threshold, sequential, diverging
- **dv_curve**: Curve interpolation - linear, step, basis, cardinal, catmull-rom, monotone, natural
- **dv_geo_core**: Geographic projections (Mercator, Equirectangular, Albers, Orthographic) and GeoJSON parsing
- **dv_delaunay**: Delaunay triangulation algorithm
- **dv_voronoi**: Voronoi diagram generation
- **dv_stats**: Statistical functions - regression, correlation, t-test, kernel density estimation
- **dv_hierarchy**: Hierarchical layouts - treemap, pack, partition, tree, cluster
- **dv_shape**: Shape generators - line, area, arc, pie, stack
- **dv_glyph**: Symbol glyphs - circle, square, triangle, diamond, cross
- **dv_gradient**: Gradient definitions - linear, radial
- **dv_pattern**: Pattern fills - dots, lines
- **dv_clip_path**: Clipping paths - rectangle, circle
- **dv_marker**: Markers - circle, arrow
- **dv_text**: Text rendering utilities
- **dv_group**: SVG-like group widget with transforms
- **dv_axis**: Chart axes - top, bottom, left, right
- **dv_grid**: Chart grid lines
- **dv_legend**: Legend widgets - ordinal, linear
- **dv_annotation**: Chart annotations - line, area, label
- **dv_geo**: Geographic visualization widgets
- **dv_event**: Pointer event handling
- **dv_tooltip**: Tooltip widget
- **dv_brush**: Brush selection - 1D and 2D
- **dv_drag**: Drag interaction
- **dv_zoom**: Zoom and pan interaction
- **dv_bounds**: Chart bounds and margins
- **dv_responsive**: Responsive container widgets
- **dv_xychart**: XY chart series - line, bar, area, scatter
- **dv_heatmap**: Heatmap visualization
- **dv_network**: Force-directed network graph
- **dv_threshold**: Reference lines and areas
- **dv_mock_data**: Mock data generators
- **data_visualization**: Umbrella package exporting all sub-packages
